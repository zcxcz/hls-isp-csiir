## 1. 功能需求概述

### 1.1 功能描述

ISP-CSIIR (Image Signal Processor - Color Space IIR) 模块是一个基于 IIR 滤波的多尺度自适应降噪模块，用于图像处理流水线中的色度分量处理。该模块通过梯度计算、多尺度方向性平均和梯度加权融合，实现边缘保持的降噪效果。

### 1.2 核心功能

1. **梯度计算与窗口大小确定**
   - 基于 5x5 Sobel 滤波器计算水平/垂直梯度
   - 根据梯度值自适应确定滤波窗口大小

2. **多尺度方向性平均**
   - 支持 2x2、3x3、4x4、5x5 四种窗口尺度
   - 支持上、下、左、右四个方向掩码

3. **梯度加权方向融合**
   - 根据各方向梯度权重进行自适应融合
   - 边界像素采用镜像处理

4. **IIR 滤波与混合输出**
   - 水平方向 IIR 混合
   - 窗口混合与最终输出

### 1.3 功能框图

```

     din_valid ---->|                  |-----> dout[9:0]

                    +------------------+
```

## 6. 处理阶段划分

### 6.1 阶段定义

模块内部处理划分为 4 个主要阶段：

```
+----------------+    +----------------+    +----------------+    +----------------+
|   Stage 1      |    |   Stage 2      |    |   Stage 3      |    |   Stage 4      |
| 梯度计算与     | -> | 多尺度方向性   | -> | 梯度加权方向   | -> | IIR滤波与      |
| 窗口大小确定   |    | 平均           |    | 融合           |    | 混合输出       |
+----------------+    +----------------+    +----------------+    +----------------+
```

### 6.2 Stage 1: 梯度计算与窗口大小确定

**输入：**
- src_uv_u10: 当前像素及 5x5 邻域

**处理步骤：**
1. 构建 5x5 滤波窗
2. 计算 Sobel 梯度 (水平、垂直)
3. 计算综合梯度 grad = |grad_h|/5 + |grad_v|/5
4. LUT 查表确定窗口大小
5. 窗口大小限幅 [16, 40]

**输出：**
- grad(i, j): 当前像素梯度值
- win_size_clip(i, j): 窗口大小

**关键运算：**
- 5x5 卷积 (乘累加)
- 绝对值运算
- LUT 查表

### 6.3 Stage 2: 多尺度方向性平均

**输入：**
- src_uv_s11: 有符号像素数据 5x5 邻域
- win_size_clip(i, j): 窗口大小

**处理步骤：**
1. 根据窗口大小选择核 (2x2/3x3/4x4/5x5)
2. 应用方向掩码 (上、下、左、右、中心)
3. 计算加权和与权重和
4. 计算方向平均值

**输出：**
- avg0_value_c, avg0_value_u/d/l/r: 小窗口各方向平均值
- avg1_value_c, avg1_value_u/d/l/r: 大窗口各方向平均值

**关键运算：**
- 条件核选择
- 多核并行计算
- 加权平均 (除法)

### 6.4 Stage 3: 梯度加权方向融合

**输入：**
- grad(i, j): 当前及邻域梯度
- avg*_value_*: 各方向平均值

**处理步骤：**
1. 获取中心及四邻域梯度
2. 边界处理 (镜像)
3. 梯度逆序排序
4. 加权融合

**输出：**
- blend0_grad(i, j): 小窗口融合结果
- blend1_grad(i, j): 大窗口融合结果

**关键运算：**
- 排序网络 (5 元素)
- 加权平均 (除法)

### 6.5 Stage 4: IIR 滤波与混合输出

**输入：**
- blend*_grad: 融合结果
- avg*_value_u: 上方平均值
- src_uv_s11_5x5: 原始数据 5x5 邻域

**处理步骤：**
1. 计算 blend_ratio_idx
2. 水平方向混合
3. 根据窗口大小选择 blend_factor
4. 窗混合计算
5. 最终混合输出
6. 限幅并转换回 U10

**输出：**
- blend_uv(i, j): 输出像素

**关键运算：**
- 多路选择
- 加权混合
- 限幅运算

### 6.6 阶段间数据流

```
           Line Buffer
               |
               v
         +-----------+
         | Stage 1   |
         | Gradient  | -> grad, win_size
         +-----------+
               |
               v
         +-----------+
         | Stage 2   |
         | Average   | -> avg*_value_*
         +-----------+
               |
               v
         +-----------+
         | Stage 3   |
         | Blend     | -> blend*_grad
         +-----------+
               |
               v
         +-----------+
         | Stage 4   |
         | Output    | -> dout_TDATA
         +-----------+
```

| 数据类型 | 位宽建议 | HLS 实现 |
|----------|----------|----------|
| ap_uint<10> | 输入/输出像素 | din_TDATA, dout_TDATA |
| ap_int<11> | 有符号中间值 | src_uv_s11 |
| ap_int<16> | 梯度值 | grad |
| ap_int<32> | 累加值 | sum, weighted_sum |

### 8.6 特殊运算处理

#### 8.6.1 除法运算

算法中存在多种除法运算：
- `avg_value = sum / sum_factor` (固定分母除法)
- `blend_grad = weighted_sum / grad_sum` (变量分母除法)

**HLS 实现建议：**
1. 固定分母除法：使用右移或倒数乘法
2. 变量分母除法：使用 HLS 数学库或迭代除法器

#### 8.6.2 排序运算

5 元素逆序排序需要：
- 约 10 个比较器
- 可组合逻辑实现

## 9. 边界处理需求

### 9.1 图像边界处理策略

| 边界位置 | 处理方式 |
|----------|----------|
| 上边界 (j=0) | grad_u = grad(i, j) |
| 下边界 (j=height-1) | grad_d = grad(i, j) |
| 左边界 (i=0) | grad_l = grad(i, j) |
| 右边界 (i=width-1) | grad_r = grad(i, j) |

### 9.2 滤波窗边界处理

5x5 滤波窗超出图像边界时，采用复制边界像素策略：
```
src_uv_u10_5x5(w, h) = src_uv_u10(clip(i+w, 0, width-1), clip(j+h, 0, height-1))
```

### 9.3 输出边界处理

输出像素值限幅到 [0, 1023]：
```
dout = clip(blend_uv + 512, 0, 1023)
```
