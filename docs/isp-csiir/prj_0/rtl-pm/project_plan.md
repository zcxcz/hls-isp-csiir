# ISP-CSIIR HLS 模块开发项目规划书

## 基本信息
- 项目名称: ISP-CSIIR HLS 模块
- 项目编号: prj_0
- 启动日期: 2026-03-23
- 设计方法: HLS (High-Level Synthesis)
- 项目经理: rtl-pm

## 项目目标

将 ISP-CSIIR 算法通过 HLS 方法实现，生成满足时序约束的可综合 RTL 代码。

### 核心交付物
- HLS C++ 源代码
- 综合后的 Verilog RTL
- 验证环境与测试报告

## 需求规格摘要

### 数据规格
| 参数 | 规格 |
|------|------|
| 输入数据 | 10-bit 无符号 (U10) |
| 输出数据 | 10-bit 无符号 (U10) |
| 中间数据 | 11-bit 有符号 (S11) |

### 性能目标
| 参数 | 目标值 |
|------|--------|
| 目标频率 | 600 MHz |
| 吞吐量 | 1 pixel/clock |
| 工艺节点 | 12nm |
| HLS工具 | Vitis HLS (默认) |

### 算法阶段
1. **Stage 1**: 梯度计算与窗口大小确定
2. **Stage 2**: 多尺度方向性平均
3. **Stage 3**: 梯度加权方向融合
4. **Stage 4**: IIR滤波与混合输出

## 技术风险识别

| 风险项 | 风险等级 | 说明 |
|--------|----------|------|
| 除法操作 | **高** | 变量除法 (avg_value, grad_sum) 需要迭代除法器或近似方案 |
| 高频时序 | **高** | 600MHz @ 12nm 需要精细流水线划分 |
| 行缓存 | 中 | 5x5窗口需要行缓存设计 |
| 条件分支 | 中 | 多阈值判断需要高效实现 |
